
How to Use this Raindrop:

1- Import Line.x in MMD
2- Import Surface.x or any other .x files that become a reference
3- Import DummyBone.pmx
4- Associate Line.x and Surface.x with Bone01 of DummyBone.pmx

- By moving Bone01, you can change the length of the Raindrop
- By moving Surface.x, you can move the raindrop without changing the length
